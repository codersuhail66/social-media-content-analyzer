Social Media Content Analyzer

This project is a web application designed to analyze social media posts and provide suggestions to improve engagement. It uses AI-powered text evaluation to help creators optimize their captions, tone, structure, and content quality for better performance across platforms like Instagram, Facebook, Twitter, and LinkedIn.

Features

AI based content analysis
Engagement improvement suggestions
Readability and tone checking
Keyword density and hashtag suggestions
Post quality score
Simple and user friendly interface
Fast, lightweight, and deployed on Netlify

How It Works

Enter your social media caption or post text
Click the analyze button
The system processes the input using AI models
Receive suggestions on improvements, hashtags, and engagement tips

Technologies Used

HTML, CSS, JavaScript
AI text analysis using an external API
Netlify for hosting and deployment
